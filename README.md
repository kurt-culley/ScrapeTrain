# ScrapeTrain
A simple TrakTrain beat downloader

# Dependencies
```
$ gem install httparty
$ gem install nokogiri
```

# Usage
```
$ ruby ScrapeTrain.rb http://traktrain.com/ARTIST_NAME
```
All files will download to ARTIST_NAME/TRACK_NAME.mp3
